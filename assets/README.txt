ASSET FILES NEEDED
==================

Please add the following image files to this folder:

1. icon.png - App icon (1024x1024 pixels recommended)
2. splash.png - Splash screen image (1242x2436 pixels recommended)
3. adaptive-icon.png - Android adaptive icon (1024x1024 pixels)
4. favicon.png - Web favicon (48x48 pixels)

You can create these images using tools like:
- Canva (free)
- Figma (free)
- Adobe Illustrator
- Online app icon generators

For a quick start, you can use placeholder images from:
https://placeholder.com or create simple colored squares.
